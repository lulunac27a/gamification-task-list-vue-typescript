# Vue gamification todo list app with Vuex, SCSS and TypeScript

## Setup the application

```
npm install
```

### Serve the application

```
npm run serve
```

### Build the application to compile and minify the code

```
npm run build
```

### Lint code to check for errors and warnings

```
npm run lint
```
